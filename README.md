# Uzima Frontend 🌿

Uzima is a health and wellness platform connecting users to healers, tasks, knowledge sharing, and rewards.

This repository contains the frontend built with **Next.js (App Router)** and **TypeScript**.

---

## 🚀 Tech Stack

- Next.js 14+
- React
- TypeScript
- Tailwind CSS
- PWA Support
- Component-driven architecture

---

## 📁 Project Structure

- `app/` → Next.js routes (pages, APIs, layouts)
- `components/` → Reusable UI components
- `components/ui/` → Design system components
- `components/tasks/` → Task system UI
- `components/healers/` → Healer marketplace UI
- `components/dashboard/` → User dashboard features
